# sus-starter-pack

Starter apps for [SUS (Single Use Software)](https://github.com/dev-dull/single-use-software).

## Getting started

1. Fork this repo
2. Point your SUS instance at your fork via the Setup page
3. Start building apps — they'll be saved to your repo

## Structure

```
{team}/
  {app-slug}/
    sus.json         # app metadata
    main.py          # entry point (or index.html for static apps)
    requirements.txt # dependencies (if Python)
```

## Included apps

- **example/hello-world** — A confetti-powered hello world page, built entirely by Claude in a SUS session
- **marketing/campaign-tracker** — Campaign tracking placeholder
- **finance/budget-dashboard** — Budget dashboard placeholder
